# CLAUDE.md

This is a skills repository — a personal library for building, testing, and sharing Claude Code skills.

## Repo layout

- `skills/` — Production-ready skills, each in its own folder with a `skill.md`
- `drafts/` — Work-in-progress skills, experiments, imports being adapted
- `evals/` — Evaluation prompts and expected outputs for testing skills
- `templates/` — Starter templates for creating new skills

## Conventions

- Every skill lives in its own folder with at minimum a `skill.md` file
- New skills start in `drafts/`, move to `skills/` once tested
- Skill prompts go in `skill.md`, documentation in `README.md`
- Keep skills focused — one clear purpose per skill
- When importing a skill from another project, drop it in `drafts/` first to adapt and test before promoting
